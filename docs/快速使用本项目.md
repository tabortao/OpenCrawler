## 启动项目

```bash
micromamba run -p ./venv python main.py
# 服务将在 http://127.0.0.1:8000 启动
```

## 使用CLI

```bash
# 提取微信公众号文章
curl -X POST http://127.0.0.1:8000/api/v1/articles -H "Content-Type: application/json" -d '{"url": "https://mp.weixin.qq.com/s/e4s2x2pqfKTlaQ4M9r3DaA", "download_images": true}'

# 提取知乎文章
curl -X POST http://127.0.0.1:8000/api/v1/articles -H "Content-Type: application/json" -d '{"url": "https://zhuanlan.zhihu.com/p/1946593133073764815", "download_images": true}'

# 提取少数派文章
curl -X POST http://127.0.0.1:8000/api/v1/articles -H "Content-Type: application/json" -d '{"url": "https://sspai.com/post/105218", "download_images": true}'

```

## 使用MCP服务器

详见[MCP服务器使用说明](mcp_server_usage.md)

### 启动MCP服务器

推荐使用HTTP模式，然后在Cherry Studio中配置MCP服务器就可以使用了。


```bash
# stdio 模式（推荐 Cherry Studio）
micromamba run -p ./venv python mcp_server.py stdio

# SSE 模式（推荐 Cherry Studio HTTP 方式），注意sse和http启动命令是不同的
micromamba run -p ./venv python mcp_server.py sse --port 8765

# HTTP 模式（推荐 Trae、Cursor），注意sse和http启动命令是不同的
micromamba run -p ./venv python mcp_server.py http --port 8765
```

## 特殊文件夹说明

### storage 文件夹

这是 **Crawlee** (Playwright 爬虫框架) 自动生成的存储目录，包含：

| 子目录              | 作用                                         |
| ------------------- | -------------------------------------------- |
| `key_value_stores/` | 键值存储，保存爬虫状态、统计数据、会话信息等 |
| `request_queues/`   | 请求队列，管理待爬取的 URL                   |

## browser_data 文件夹

这是 **浏览器数据目录**，用于存储：

- Cookie 和登录状态
- 浏览器缓存
- 用户会话数据
