## 启动项目

```bash
micromamba run -p ./venv python main.py
# 服务将在 http://127.0.0.1:8000 启动
```

## 使用

```bash
# 提取微信公众号文章
curl -X POST http://127.0.0.1:8000/api/v1/articles -H "Content-Type: application/json" -d '{"url": "https://mp.weixin.qq.com/s/e4s2x2pqfKTlaQ4M9r3DaA", "download_images": true}'

# 提取知乎文章
curl -X POST http://127.0.0.1:8000/api/v1/articles -H "Content-Type: application/json" -d '{"url": "https://zhuanlan.zhihu.com/p/1946593133073764815", "download_images": true}'

# 提取少数派文章
curl -X POST http://127.0.0.1:8000/api/v1/articles -H "Content-Type: application/json" -d '{"url": "https://sspai.com/post/105218", "download_images": true}'

# 提取 GitHub 仓库文章
curl -X POST http://127.0.0.1:8000/api/v1/articles -H "Content-Type: application/json" -d '{"url": "https://github.com/tabortao/gocron", "download_images": true}'

# 提取小红书文章
curl -X POST http://127.0.0.1:8000/api/v1/articles -H "Content-Type: application/json" -d '{"url": "http://xhslink.com/o/50GU2ftTs5j ", "download_images": true}'

```
